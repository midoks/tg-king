### 简介

为定制TG功能而生,自由开发。

* BOT/Client管理
* 日志管理
* 模块方式管理

### 初始安装使用

```
curl --insecure -fsSL  https://raw.githubusercontent.com/midoks/tg-king/main/scripts/install_dev.sh | bash
```
